# Приемник монет ccTalk (1, 3, 5 сомов)

Программа для приема монет номиналом 1, 3 и 5 сомов с помощью ccTalk монитоприемника nrig13.

## Функциональность

- Автоматический поиск COM порта монитоприемника
- Графический интерфейс (GUI) на PyQt5
- Отображение статуса подключения
- Подсчет принятых монет по номиналам
- Автоматическое срабатывание механизма обратного выброса:
  - 1 сом → 1 минута
  - 3 сома → 3 минуты
  - 5 сомов → 5 минут

## Требования

- Python 3.8+
- PyQt5
- pyserial

## Установка

```bash
pip install -r requirements.txt
```

## Использование

```bash
python main.py
```
